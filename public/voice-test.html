🔊 Voice Test - ByajBook

If you can't hear voice in the app, follow these steps:

📋 Before Testing - Check These:

  1. Device Volume: Turn up your device volume
  2. Silent Mode: Turn OFF silent/vibrate mode
  3. Browser: Try Chrome or Safari (not Firefox)
  4. Headphones: If using headphones, check they're connected properly
  5. macOS: System Preferences → Sound → Output → Check your speakers

What should happen: You should hear "Hello. This is an emergency voice test. Can you hear me?"

If you don't hear anything: Check the log below for error messages.

🔧 Manual Browser Console Test

If the button doesn't work, try this in browser console (F12 → Console):

window.speechSynthesis.speak(new SpeechSynthesisUtterance('Hello test'))

Copy and paste this line in your browser's developer console and press Enter.
